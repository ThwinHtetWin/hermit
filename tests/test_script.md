## Purge Hermit

To start from a clean install, remove any files that hermit looks for
or creates:

```
rm /tmp/shard_words.bson
rm /tmp/shard_words.bson.persisted
rm /tmp/shard_words.bson.backup
rm /etc/hermit.yaml
```

## Test Interface

```
source environment.sh

```
